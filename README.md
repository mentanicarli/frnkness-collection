# Pupsiks Saga

Минималистичный музыкальный веб-плеер с коллекцией релизов, синхронизированными текстами и статистикой прослушиваний.

Проект создан для личного использования и не является стриминговым сервисом.

## Демо

https://mentanicarli.github.io/pupsiks-saga/

## Основные возможности

### Плеер

- Мини-плеер с быстрым доступом
- Полноэкранный режим воспроизведения
- Переключение треков внутри релиза
- Режим «Поток» для непрерывного случайного воспроизведения
- Глобальный поиск по трекам, альбомам и текстам


### Тексты песен

- Поддержка .txt и .lrc форматов
- Переключение между текстом и режимом караоке
- Синхронная подсветка строк с автоматической прокруткой
- Оптимизированная прокрутка для плавного воспроизведения

### Статистика

- Чарт треков по количеству прослушиваний
- Счетчик прослушиваний для каждого альбома
- Хранение данных через Supabase

### Интерфейс

- Динамическая цветовая тема на основе обложки
- Анимации и плавные переходы
- Минималистичный интерфейс
- Адаптивный дизайн для всех устройств

### PWA

- Установка на iOS через Safari (Add to Home Screen)
- Поддержка Web App Manifest
- Работа в standalone-режиме
- Оптимизация для мобильных устройств

## Архитектура

Проект прошел миграцию от статического HTML к компонентной архитектуре.

Текущий стек:

- Vue 3 - UI и реактивность
- TypeScript - типизация и строгая проверка
- Pinia - централизованное управление состоянием
- Vite - сборка и dev-сервер
- Tailwind CSS - стилизация
- Supabase - backend для счетчика прослушиваний
- Color Thief - генерация цветовой темы
- PWA - installable web app

## Быстрый старт

### Требования

- Node.js 18+ (рекомендуется 20)

### Установка

```bash
npm install
npm run dev
```

Создайте файл .env.local на основе .env.example перед запуском.

### Сборка

```bash
npm run build
npm run typecheck
npm run preview
```

Сборка будет доступна в папке dist.

## Деплой (GitHub Pages)

Проект настроен на автодеплой через GitHub Actions.

Шаги:

1. Запушить код в ветку main
2. Перейти в Settings -> Pages
3. Выбрать GitHub Actions как Source
4. Дождаться выполнения workflow

## Ограничения

- Проект не рассчитан на масштабирование
- Нет системы авторизации
- Supabase используется только для счетчика прослушиваний
- Часть логики находится в legacy-слое

## Конфигурация Supabase

Для работы чарта требуется:

- Таблица с треками
- RPC-функция increment_play_count

Рекомендуется вынести URL и API KEY в переменные окружения (.env).

Используется конфигурация через Vite env:

- VITE_SUPABASE_URL
- VITE_SUPABASE_ANON_KEY
- VITE_SHOW_NEW_RELEASE_PROMO
- VITE_NEW_RELEASE_PROMO_ID
